# vimium-theme

Personal Vimium (Chrome extension) settings.

## Files

- CSS: [vimium/custom.css](vimium/custom.css)
- Key mappings: [vimium/keymap.txt](vimium/keymap.txt)
- Optional export backup: [vimium/options-export.json](vimium/options-export.json)

## Apply

Open Vimium Options and paste:

- [vimium/custom.css](vimium/custom.css) → "Custom CSS for Vimium UI"
- [vimium/keymap.txt](vimium/keymap.txt) → "Custom key mappings"
